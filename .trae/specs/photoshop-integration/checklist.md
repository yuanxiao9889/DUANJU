# Checklist

## 阶段一：本地 HTTP 服务器

- [x] HTTP 服务器能在指定端口正常启动
- [x] 端口冲突时能自动查找可用端口
- [x] `/api/ps/image` 端点能正确接收 Base64 图像数据
- [x] `/api/ps/status` 端点能返回服务器状态

## 阶段二：图片接收提示气泡

- [x] 收到图片时能在界面右下角显示提示气泡
- [x] 气泡中能正确显示图片预览
- [x] 点击"添加到画布"能在画布中心创建图片节点
- [x] 气泡超时（30秒）后能自动关闭
- [x] 支持多图片队列显示

## 阶段三：发送图片到 Photoshop

- [x] 能从图片节点发送图片到 PS 插件
- [x] PS 未连接时显示正确的错误提示
- [x] 发送成功/失败有明确的状态反馈

## 阶段四：设置页面配置

- [x] 设置页面有 PS 联动配置区域
- [x] 能配置服务器端口
- [x] 能启用/禁用 PS 联动功能
- [x] 显示当前连接状态

## 阶段五：PS 插件示例

- [x] PS 插件能正确安装到 Photoshop
- [x] 插件能获取 PS 当前选区图像
- [x] 插件能发送图像到本软件
- [x] 插件能接收图像并填充到 PS 选区
- [x] 有完整的使用说明文档

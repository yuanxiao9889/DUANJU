# 简化版剧本导入功能计划

## 一、需求分析

简化导入剧本功能：
1. 用户点击"导入剧本"
2. 弹出对话框询问章节数量
3. 生成空的章节节点，用户自己填写内容

**注意**：提取资产功能已在资产面板实现，不需要重复。

## 二、实现方案

### 导入流程

```
点击导入 → 输入章节数量 → 创建根节点 + N个空章节节点
```

### 生成内容

- 1 个剧本根节点（标题节点）
- N 个空章节节点（用户自己填写内容）

## 三、修改文件

### 3.1 新增文件
- `src/features/canvas/ui/ChapterCountDialog.tsx` - 章节数量输入对话框

### 3.2 修改文件
- `src/features/canvas/ui/ScriptBiblePanel.tsx`
  - 移除文件选择器
  - 移除 LLM 调用
  - 移除资产节点创建
  - 简化为只创建空章节节点

## 四、UI 变更

### ScriptBiblePanel 工具栏

```
[导入剧本] [导出]
```

- **导入剧本**：弹出对话框，输入章节数量，创建空章节节点
- **导出**：保持不变

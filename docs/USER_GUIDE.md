## 1. 产品简介

**分镜助手** 是一款桌面端 AI 辅助分镜制作工具，主要功能包括：

- **可视化节点画布**：通过拖拽节点构建分镜/剧本结构
- **AI 图像生成**：集成多种 AI 图像生成提供商
- **一站式工作流**：从构思到导出，全流程支持
- **双模式支持**：分镜项目和剧本项目

### 技术栈

- 前端：React 18 + TypeScript + Zustand + `@xyflow/react` + TailwindCSS
- 桌面容器：Tauri 2
- 后端：Rust
- 数据存储：SQLite

## 2. 安装与运行

### 2.1 环境要求

- Node.js 20+
- npm 10+
- Rust stable（含 Cargo）
- Tauri 平台依赖

### 2.2 安装步骤

```bash
# 1. 克隆或下载项目
cd storyboard-copilot

# 2. 安装依赖
npm install
```

### 2.3 运行方式

**仅前端开发模式：**
```bash
npm run dev
```

**Tauri 联调模式（推荐）：**
```bash
npm run tauri dev
```

### 2.4 构建发布

```bash
# 构建桌面应用
npm run tauri build
```

构建完成后，安装包位于：
- Windows: `src-tauri/target/release/bundle/msi/` 或 `src-tauri/target/release/bundle/nsis/`

## 4. 快速开始

### 4.1 创建新项目

1. 启动应用后，点击顶部工具栏的 **新建项目**
2. 选择项目类型：
   - **分镜项目**：用于图像分镜制作
   - **剧本项目**：用于剧本/故事创作
3. 输入项目名称，点击确认

### 4.2 添加节点

1. 从左侧面板拖拽节点到画布
2. 或右键画布空白区域，选择「添加节点」
3. 常用节点：
   - **分镜节点**：基础分镜单位
   - **剧本节点**：故事结构单元
   - **AI 生成节点**：调用 AI 生成内容

### 4.3 连接节点

1. 从节点锚点拖拽出线
2. 连接到目标节点的锚点
3. 支持多连线、分支连线

### 4.4 使用 AI 生成图像

1. 添加 AI 生成节点
2. 在底部面板输入提示词
3. 选择 AI 提供商和模型
4. 点击生成

## 5. 核心功能

### 5.1 节点操作

| 操作 | 说明 |
|------|------|
| 添加节点 | 拖拽或右键菜单 |
| 删除节点 | 选中后按 Delete 或右键删除 |
| 移动节点 | 拖拽节点 |
| 缩放节点 | 选中节点，拖拽边缘手柄 |
| 复制节点 | Ctrl+C / Ctrl+V |
| 粘贴节点 | Ctrl+V |
| 全选 | Ctrl+A |
| 撤销 | Ctrl+Z |
| 重做 | Ctrl+Shift+Z |

### 5.2 框选与批量操作

- **框选**：鼠标拖拽框选多个节点
- **批量连线**：框选多个节点后，可从合并锚点拖出分支连线
- **批量删除**：框选后按 Delete

### 5.3 对齐辅助

拖动节点时，画布会显示对齐虚线辅助对齐。

### 5.4 导出功能

支持多种导出格式：

| 格式 | 说明 |
|------|------|
| PNG/JPG | 单张图片导出 |
| PDF | 文档导出 |
| DOCX | Word 文档 |
| JSON | 项目数据 |

## 6. 节点系统

### 6.1 分镜节点

- **分镜节点**：基础分镜单元，包含图像和描述
- **图像节点**：纯图像内容
- **文本节点**：纯文本描述

### 6.2 剧本节点

- **剧本根节点**：整个剧本的根节点
- **章节节点**：章节结构
- **角色节点**：角色信息
- **场景节点**：场景描述
- **道具节点**：道具信息
- **情节点**：情节单元

### 6.3 AI 节点

- **AI 生成节点**：调用 AI 生成图像
- **AI 对话节点**：与 AI 对话交互

## 10. 常见问题

### 10.1 安装问题

**Q: npm install 失败**
- 确保 Node.js 版本 >= 20
- 尝试清理缓存：`npm cache clean --force`

**Q: Rust 编译错误**
- 确保已安装 Rust：`rustup install stable`
- 更新 Rust：`rustup update`

### 10.2 运行问题

**Q: Tauri 启动失败**
- 检查 Tauri 平台依赖是否安装
- 查看控制台错误信息

### 10.3 AI 生成问题

**Q: AI 生成失败**
- 检查 API Key 是否正确配置
- 确认网络连接正常
- 查看配额是否用完

## 附录

### A. 项目结构

```
src/
  features/canvas/          # 画布主流程
  features/project/         # 项目管理
  features/settings/        # 设置功能
  stores/                   # 全局状态
  commands/                 # Tauri 命令桥接
  i18n/                     # 国际化

src-tauri/
  src/commands/             # Rust 命令实现
  tauri.conf.json           # Tauri 配置
```

### B. 数据存储

- SQLite 数据库位置：`app_data_dir/projects.db`
- 表结构：
  - `projects`: 项目数据
  - `nodes_json`: 节点数据
  - `edges_json`: 连线数据
  - `viewport_json`: 视口数据

### C. 国际化

支持中文和英文，可在设置中切换语言。

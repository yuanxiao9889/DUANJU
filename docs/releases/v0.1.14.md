# v0.1.14

## 新增

- 多节点框选批量连线功能：框选多个节点后可从合并锚点拖出分支连线
- 节点对齐辅助线：拖动节点时显示对齐虚线
- 剧本节点系统：添加剧本根节点、章节节点、角色节点、场景节点、道具节点、情节点
- AI 提供商扩展：添加阿里云、Comfly、真真等提供商支持
- 项目类型选择：支持分镜项目和剧本项目两种类型
- 剧本导出功能：支持导出为 TXT、DOCX、JSON、Markdown 格式
- 大纲生成功能：AI 自动生成故事大纲
- 资产提取功能：从剧本中提取角色、场景、道具等资产

## 优化

- 优化画布交互体验
- 优化节点拖拽性能
- 优化项目创建流程

## 修复

- 修复 TypeScript 编译错误
- 修复 Map 类型冲突问题
- 修复部分 UI 显示问题

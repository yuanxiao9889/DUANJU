## 新增

- Windows 版本接入 Tauri 官方应用内更新，支持在应用内检查、下载并安装新版本。
- 更新弹窗支持展示版本号、发布说明、下载进度、安装状态，并保留 GitHub Release / 手动下载兜底入口。

## 优化

- 统一启动自动检查更新与设置页手动检查的更新链路，非 Windows 平台继续走手动下载入口。
- 调整 GitHub Release 打包流程，Windows 产出安装包、签名文件与 `latest.json`，macOS 持续产出 DMG。
- 修正 GitHub Actions 中 Windows updater 元数据上传参数，恢复自动发布链路。

## 修复

- 修复旧版仅能跳转 GitHub 下载、无法直接应用内升级的问题。
- 修复更新检查失败时阻塞使用的问题，失败后会保留手动下载兜底。
- 首个支持应用内更新的版本仍需用户手动安装一次，安装完成后后续版本即可在应用内更新。

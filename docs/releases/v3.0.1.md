## 新增

- 新增“电商”项目入口与电商出图工作区，支持商品资料、出图 Brief、批量出图和结果组阶段节点。
- 新增右侧电商 Agent 面板，支持上传商品图、手动同步商品信息、反推商品信息和边聊边更新左侧画布节点。
- 新增电商 Agent 引导卡片，展示已理解内容、待确认信息、推荐方向、下一步问题和快捷补充，支持多选后填入输入框。
- 新增电商批量出图参数，支持按厂商选择图像模型、动态选择分辨率、配置比例和每比例张数。

## 优化

- 上传商品图后不再自动反推，改为用户点击“反推商品信息”后再调用多模态 LLM。
- 商品信息明确前隐藏生成参数区，避免用户在产品理解不足时直接进入生成。
- 电商出图默认图像厂商使用 `oopii-推荐`，默认模型使用 `oopii/gpt-image-2`。
- 电商阶段节点提高默认高度并支持自由缩放，商品图直接显示在商品资料卡内。

## 修复

- 修复商品图预览和生成参考图链路不清晰的问题，生成时优先使用商品资料卡中的商品图作为参考图。
- 修复普通文本模型识别商品图时缺少明确多模态提示的问题。

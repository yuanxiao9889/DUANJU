## 新增

- Windows 版本接入 Tauri 官方应用内更新，支持在应用内检查、下载并安装新版本。
- 更新弹窗支持展示版本号、发布说明、下载进度、安装状态，并保留 GitHub Release / 手动下载兜底入口。

## 优化

- GitHub Release 工作流调整为先构建 macOS，再在 macOS 成功后继续构建 Windows。
- CI 打包阶段统一规范 bundle 产品名，降低 macOS 与 GitHub Release 资产生成时的兼容性问题。
- 对齐桌面端 updater / process 插件装配方式，并补齐 macOS updater 归档解包所需依赖。

## 修复

- 修复 GitHub Actions 中 updater 元数据上传、签名密钥注入和 Tauri 插件版本不一致导致的发布失败。
- 修复 macOS 打包阶段因 capability 与插件装配范围不一致、updater 归档依赖缺失而中断的问题。
- 首个支持应用内更新的版本仍需用户手动安装一次，安装完成后后续版本即可在应用内更新。

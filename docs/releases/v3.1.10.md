## 新增

- 新增 macOS 打包工作流，支持通过 GitHub Actions 生成 macOS 安装包。
- 导演舞台新增相机路径录制、时间线缩放与 MP4 导出能力。
- 商业广告项目补充批量生成、素材配置与视频生成相关能力。

## 优化

- 优化画布保存、启动恢复和退出持久化流程，降低大项目卡顿。
- 优化 AI 图片生成结果的编组继承、连接点显示和拖入拖出编组体验。
- 优化图片生成请求的尺寸保持逻辑，避免部分模型别名丢失目标尺寸。
- 优化设置与供应商模型配置界面，补充 Seedance 与 GPT 视频参数。

## 修复

- 修复组内生成结果再次生成时跑到编组外的问题。
- 修复生成结果节点连接点在缩放或编组场景下偏移的问题。
- 修复节点拖出编组失败、拖入编组后移动整组遗漏旧节点的问题。

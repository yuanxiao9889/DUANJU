## 新增

- 在设置的关于页新增错误日志入口，支持查看、筛选、复制诊断 ID、删除、清空并导出近 7 日错误日志表格。
- 图片生成提交失败和轮询失败会记录本地错误日志，保留 14 天，并记录 Request ID、Trace ID、任务 ID、模型和错误分类等排障字段。

## 优化

- NewAPI 图片生成失败时会尽量从响应头和响应体提取 Request ID，方便在后台快速定位错误记录。
- SeedVR2 等离线扩展支持自定义启动健康检查超时，避免冷启动依赖加载较慢时被误判失败。

## 修复

- 修复扩展启动失败时只显示通用错误文案的问题，现在会显示后端返回的真实失败原因。

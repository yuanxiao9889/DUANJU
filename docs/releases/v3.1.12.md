## 新增

- 新增「AI造型师:高转化模特图」技能，支持无商品图时根据文字需求生成模特造型方向。
- 为 AI 造型师增加背景决策选项，可选择白色影棚背景或按造型风格匹配场景背景。
- Agent 面板支持折叠为右侧竖向对话气泡，便于释放画布空间。

## 优化

- 优化广告创意和通用 Agent 的多轮推荐逻辑，第二轮后自动进入方案节点，后续选择会更新方案。
- 优化 AI 造型师批量出图提示词，同一风格下生成不同模特、姿态、镜头和场景变化。
- 优化新对话任务隔离，避免新任务继承上一轮参考图、方案节点或图片上下文。
- 优化方案摘要展示，长内容限制高度并支持内部滚动。

## 修复

- 修复无 skill 引导选项被通用模板替代的问题，改为依赖 LLM 定制推荐。
- 修复点击新建对话时输入框草稿被清空或未真正创建新任务的问题。

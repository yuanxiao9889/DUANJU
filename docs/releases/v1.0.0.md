## 新增

- 新增场景工作台、分集生成与 Script Reference / Asset Reference 工作流，扩展剧本到分镜的连续创作链路。
- 图片生成节点与即梦图片节点新增相机参数选择器，支持隐藏注入镜头参数、快捷预设和参数说明弹窗。
- 参数说明弹窗接入焦距、镜头、光圈、相机四类教程图，并支持分栏切换浏览。

## 优化

- 优化相机参数选择器的交互动画、图标态、高亮状态和说明弹窗布局，提升参数浏览与选择体验。
- 优化 Dreamina / Jimeng 提示词与提交流程，补强参考素材、对话文本和多模态场景下的稳定性。
- 优化 GitHub Release 构建流程，Windows 继续发布正式 NSIS 安装包，macOS 在无 Apple 签名材料时发布带 `unsigned-test` 标识的内部测试 DMG。

- 修复图片生成、兼容供应商与 Nano Banana 相关模型请求的多处兼容性问题。
- 修复 Dreamina CLI 更新、登录回退、回调端口与任务提交过程中的若干稳定性问题。
- 修复画布、节点持久化、资源路径与媒体加载中的多项问题，降低项目恢复和引用失败概率。

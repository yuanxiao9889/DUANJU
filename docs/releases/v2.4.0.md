## 新增

- 剧本画布新增“故事参考”节点，可手写全局故事概括、设定约束和连续性参考，并自动参与章节扩写、场景分集、Scene Copilot 与连续性校验。
- 剧本欢迎窗新增“跳过并进入空画布”，跳过状态按项目持久化，空剧本项目不再反复弹出欢迎窗。
- 脚本引用节点新增全选与取消全选能力，批量处理章节引用更方便。

## 优化

- 设置页信息结构调整，将应用更新相关内容并入关于页，并移除独立更新记录入口。
- AI 图片生成结果节点增加“正在提交请求”阶段，提交成功后再进入生成等待状态。
- 图片与视频生成节点支持由上游文本接管 prompt，接入文本后本地输入区以遮罩锁定。
- 新增本地打包提示词规则文档，方便后续 GitHub 打包与发布流程复用。

## 修复

- 修复部分图片生成请求在本地提交阶段卡住时，画布提前进入假等待并最终超时的问题。
- 修复参考图解析、远程图片读取和图片持久化链路缺少明确超时导致请求未真正发出的风险。
- 修复 macOS 多图拖入、图片映射和切割图片场景下可能只保留预览图或卡住的问题。
- 修复 OOPII / NewAPI 图片接口兼容路径，提升代理与图片生成、编辑接口的兼容性。
